How do I play ReGrowtopia?
Great question! Here's what you'll need to play ReGrowtopia.
I want an easy way to play!

The ReGrowtopia Client is an easy way to play ReGrowtopia on any device without using real Growtopia clients. It comes with a handful of custom features such as custom weathers (they render as default in real GT), and others.

Download the ReGrowtopia launcher for Windows
Download the ReGrowtopia APK for Android
I want to play with the real Growtopia client!

To play with real Growtopia clients, use one of the methods below:
Android:
 Download the hosts file. (Use Virtual Hosts or Hosts Go)

iOS:
 Use the following domain settings in Surge 5:
DOMAIN: growtopia1.com VALUE: growmmfi.de
DOMAIN: growtopia2.com VALUE: growmmfi.de
DOMAIN: rtsoft.com VALUE: growmmfi.de
DOMAIN: hamumu.com VALUE: growmmfi.de

That, or click "Download Profile from URL" and paste the following link: http://regrowtopia.com/regt.conf

Windows:
 On Windows, paste this into your hosts file -- (%WINDIR%\System32\drivers\etc) (and remove the rest):
51.75.76.141 growtopia1.com
51.75.76.141 growtopia2.com
51.75.76.141 rtsoft.com
51.75.76.141 hamumu.com

macOS:
 On macOS, paste this into your /etc/hosts file (and comment out the rest):
51.75.76.141 growtopia1.com
51.75.76.141 growtopia2.com
51.75.76.141 rtsoft.com
51.75.76.141 hamumu.com
